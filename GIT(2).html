git fetch origin           将服务器仓库中的东西下下来（分支和代码）
git co 分支名          切换分支（可以跟踪远端的分支）
git pull==git fetch+git merge
git br -a                查看跟踪的所有内容
git pull origin master(本地分支名) : master(远端分支名)
git rm -r ——cached  文件名     删除远端的文件
git rebase （给予分支）（重做分支）          
git rebase ——continue       当冲突时，处理完冲突执行该操作
